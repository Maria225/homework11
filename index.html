Latitude vs X

In this project, weather changes vs latitude location were analyzed.

First, weather data from OpenWeatherMap API were pulled for over 500 cities.

Second, the weather changes vs latitude location were plotted into graphs using matplotlib in jupyter notebook.

The weather changes analyzed were maxiumum temperture, cloudiness, wind speed, and humidity.

This website contains the source data gathered, the visualziations created, and breif description of the trends identified.
